# SampleSuperStore
Data Analysis project using Excel

## Project Overview

This project analyzes an E-commerce Sales dataset using Excel and Power BI.

## Objectives

- Data Cleaning
- Exploratory Data Analysis (EDA)
- KPI Calculation
- Sales Analysis
- Profit Analysis
- Dashboard Creation

## Tools Used

- Microsoft Excel
- Pivot Tables
- Charts

## Key KPIs

- Total Sales
- Total Profit
- Total Quantity
- Profit Margin

## Dashboard

Dashboard screenshot is included in this repository.

## Key Insights

- West region generated the highest sales.
- Technology category generated the highest profit.
- Phones were the top-selling products.
- Furniture showed lower profitability.
- Overall Profit Margin is 12.47%.
